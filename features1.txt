- Product Name: AssetTrackPro
- Web-based system assumed
- Added asset history tracking
- Included login system

- Each employee has unique ID
- Assets assigned one at a time
- Admin manages asset allocation
- Data stored securely in database

1. Employee Management
   Add/edit/remove employee details.

2. Asset Management
   Add asset details like serial number, type, condition.

3. Asset Assignment
   Assign asset to employee and record date.

4. Asset Return Tracking
   Record asset return with condition update.

5. Search Feature
   Find asset by employee or asset ID.

6. Reports
